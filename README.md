# Excel Data Cleaning Tool (Python)

## 🚀 Overview
This project is a Python-based tool that automatically cleans Excel/CSV data using pandas.

## 🔹 Features
- Removes duplicate rows
- Handles missing values
- Cleans and formats data
- Exports a new cleaned file

## 🛠 Tech Used
- Python
- pandas

## ▶️ How to Run
1. Install dependencies:
   pip install pandas

2. Run the script:
   python main.py

3. Enter your CSV file name

## 📊 Example
Input: sample_input.csv  
Output: clean_data.csv

## 💼 Use Cases
- Business data cleaning
- Excel automation
- Student projects
